# 🛡 Sentinel AI

AI-Powered Cyber Threat Detection Platform

Sentinel AI is a cybersecurity application that detects phishing URLs and analyzes suspicious screenshots using Machine Learning and Google Gemini AI. The system provides threat classification, risk scores, and security recommendations to help users identify online threats.

---

## 🚀 Features

- 🔗 Phishing URL Detection
- 🖼 Screenshot Analysis
- 🤖 Google Gemini AI Integration
- 📊 Threat Score
- 🎯 Threat Classification
- 🛡 Security Recommendations
- ⚡ Fast Real-Time Analysis
- 💻 Clean Flask Web Interface

## ⚙ Installation

```bash

```

```bash

```

Run the project

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

---

## 📌 How It Works

1. Enter a suspicious URL or upload a screenshot.
2. Sentinel AI extracts security features.
3. Machine Learning predicts whether the URL is safe or malicious.
4. Google Gemini AI analyzes the content and explains the possible threats.
5. The application generates a threat report with recommendations.
